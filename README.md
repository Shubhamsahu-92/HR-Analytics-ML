# HR - Attrition Analytics -  Exploratory Analysis & Predictive Modeling

### DATA ATRRIBUTES

satisfaction_level: Employee satisfaction level <br>
last_evaluation: Last evaluation  <br>

department: Department <br>
salary: Salary <br>
